📋 Вставка ссылок

Автоматическое распознавание и вставка URL из буфера

⚠️ Внимание! Вы открыли файл через file:// протокол.
Браузеры блокируют автоматическое чтение буфера для локальных файлов.

✅ Как исправить:
1. Используйте Ctrl+V в поле ввода (это работает всегда)
2. Или запустите локальный сервер:
# Python: python -m http.server 8080
# Node: npx http-server
# VS Code: установите "Live Server"
🎯 Способ 1: Ctrl+V (работает всегда)
Скопируйте любую ссылку и нажмите Ctrl+V (или Cmd+V на Mac) в поле ниже.
Ссылка будет автоматически распознана и вставлена.

🚀 Способ 2: Альтернативный метод (требуется сервер)
Если вы запустите страницу через локальный сервер (не file://), то будет работать
автоматическое отслеживание буфера — достаточно скопировать ссылку, и она сама вставится!
# Как запустить сервер:
• Python 3: python -m http.server 8080
• Python 2: python -m SimpleHTTPServer 8080
• Node.js: npx http-server
• VS Code: установите расширение "Live Server"
# Затем откройте http://localhost:8080
